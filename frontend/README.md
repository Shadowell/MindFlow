# MindFlow Frontend

Static Vite + React + TypeScript prototype for the MindFlow AI content creation workbench.

## Scripts

```bash
npm install
npm run dev
npm test
npm run build
npm run lint
```

## Current Scope

- Hot topic radar with mock trend data.
- Persona selection for account voice.
- Image-text draft composer with mock generation.
- Douyin, Weibo, and Xiaohongshu preview tabs.
- Schedule queue panel with mock publish state.

## Not Connected Yet

- PostgreSQL persistence.
- AI provider calls.
- Douyin, Weibo, or Xiaohongshu APIs.
- `/Users/jie.feng/wlb/AutoRepost` publishing integration.
